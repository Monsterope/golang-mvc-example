# Example Project Go lang model MVC

Project tutorial, learning for golang model MVC

## Recheck have start services
- Go-Version: 1.24
- Database: MYSQL
- Cache: Redis

## Install Project
- Init project
```
go mod download
```

- Run project
```
go run main.go
```
